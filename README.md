# Vro Easy Missiles

**Author**: Odag  
**Version**: 1.0.0

## Description
 Missiles seems like they face alot of struggles when it comes to their ability to be counter messasured,the nuances of aux ships resupply mechanics and the limited carrying capacity. 

 this mod attemps to easy some of the by reducing missile size to 1 and cost to 1 Energy Cell and 1 energy credit.

## Requirements
- X4: Foundations
- Variety and Rebalance Overhaul (VRO)

## Installation
1. Download the latest release.
2. Extract the `Vro-Easy-Missiles` folder into your X4 Foundations `extensions` directory.
   - Windows: `C:\Program Files (x86)\Steam\steamapps\common\X4 Foundations\extensions`
   - Linux: `~/.steam/steam/steamapps/common/X4 Foundations/extensions`
